## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

Boilerplate containing

- swagger for API docs
- seeder for populating data
- plop for quick template

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run dev

# production mode
$ npm run start:prod
```

## Running the Mongodb database

```bash
# start the mongodb using docker
$ docker compose up
```

## Seed data

```bash
# run seeder normally
$ npm run seed

# run seeders and replace existing data
$ npm run seed:refresh
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
